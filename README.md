# csb-examen-aut-2026
Repositorio destinado a centralizar y gestionar los exámenes técnicos de automatización para nuevos recursos QA, incluyendo ejercicios prácticos de pruebas Web, APIs y Mobile. Su objetivo es evaluar conocimientos, buenas prácticas, capacidad de análisis, diseño de escenarios, implementación de scripts automatizados, manejo de evidencias, etc..
